Programming Basic Concepts

  1. Operators

    These are primarily mathematical operators. However some languages allow the the programmer to implement them to work with more complex variables.

    • + (Addition)

      Addition operator is universally used for string concatination aswell. 

                    "hello" + " " + "world" == "hello world"

    • - (Subtraction)

    • * (Multiplication)

    • / (Division)

    • = (Assignment)

      The assignment opperator is used to set the value of a variable

    • == (Equality)

      Equality is almost allways and operation that can be defined for more complex variables.

    • ! (Not)

      Not negates any boolean expressions: 
                      !boolean - true and false are reversed
                != - not equal to

    • & (And)

                    this & that - this and that (test both)
              this && that && somthing else - stop test at first false expression

    • | (Or)

                    this | that - this or that (test both)
              this || that || somthing else - stop test at first true expression

  2. Variables

    Variables simply represent data of various types.

    • Null

      Nothing

    • Boolean

      True/False

    • Number

    • String

      a sequence of characters

    • Object

      Found only in Object oriented languages and can represent anything from any of the above data types to something as complex as a nasa space shuttle.

  3. Conditional

    Change the direction of the code based on a boolean value

    • if/else

      If else statments are universially availible in all languages. And the most common way to direct the flow of a program. 

                    if (going to walmart) {
                go right
              } else if (going to whole foods) {
                go left
              } else {
                cause a traffic jam
              }

    • switch

      switch statments are availible in most langagues and can make your code much easier to read. 

                    switch (going to) {
                dinner: take the tesla
                work: take the truck
                fishing: take the boat
                christmas: take the rv
              }

  4. Loops

    Repetitive cycles of code consisting of:
    (an initial value; a test; a limit)

    • For

      Availible in most langagues and it is great for simple itterative processes.

                    for(index = 0; index < 100; index++) {
                print(index)
              }

    • While

      Good to use when the itteration step is more complex. 

                    iterator = parent;
              while (iterator.value != searchValue) {
                if (iterator.value > searchValue) {
                  iterator = iterator.leftChild();
                } else {
                  iterator = iterator.rightChild();
                }
              }

    • Do While

      Can make your code easier to read in rare cases where you need to run the loop atleast once without running the test; 

                    cmd = null;
              do {
                cmd = (enter command)
                run(cmd)
              } while (cmd != esc)

  5. Functions

    Primary tool for abstracting out complexity from your programs, making it easier to read, understand and fix. For you aswell as others.

    A function in high level languages always has atleast 3 parts 

                name - a referance value, basically a varible that corresponds to a function.
            arguments - variables to be passed in to the function.
            return - varible/s to be returned from the function.
          
                [name](arguments){
              return [value]
            }

    Note: Assembly languages function calls are a little more complex. If you want a video on the basics of assembly languages you can vote here.
