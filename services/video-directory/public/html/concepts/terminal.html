Terminal Commands

Beginner

  1. cd

    Change directory 

              cd ./directory
          #Go back twice
          cd ../..
          #Root directory
          cd /
          cd ~/directory/in/user/home

  2. ls

    list files/directories 

              ls -a

  3. rm

    Remove 

              #Recursive/Force
          rm -f ./file-that-is.protected
          rm -rf ./directory/

  4. xdg-open

    Open a file with the default application 

              xdg-open [file/or/directory]

  5. mkdir

    Make a directory 

              mkdir -p ./none/of/these/folders/exist
          #But now they do....

  6. touch

    create a file if it does not exist 

              touch ./file.name

  7. cat

    Print contents of file 

              cat ./a.file

  8. chmod

    Change file permissions 

              chmod +w ./add-write.access
          chmod +r ./add-read.access
          chmod +x ./add-exicution.access

  9. echo

    Print the arguments/often used to pipe data into another command. 

              #-e causes echo to resolve backslash characters \n \t
          echo -e "line one\nline 2\nline III"
          echo -n "no new line appended"
          echo "garbageHello Worldgarbage" | grep -oP "H.*d"

  10. unzip

    Unzip.... 

              unzip ./zip.file

  11. zip

    Zip... 

              Zip ./zip.file
          Zip -r ./directory/


"Advanced"

  1. find

    search for files 

              #Files only with matching name
          find . -name "./*/wild*cards*" -type f
          #Directories only not recursive
          find . -maxdepth 1 -type d
          #Exclude a file path
          find -not -path "*/junk*"

  2. grep

    Search text 

              #Search a file
          grep "search text" ./file.txt
          #Search files
          grep -r "search text" ./directory/
          #search with pearl regex
          grep -P "search.*text" ./file.txt
          #Show match only (only useful with regex... I usually use it recursivly)
          grep -r -oP "search.*text" ./directory/
          #No file name
          grep -h "search text" ./file.txt
          #File name only
          grep -l "search text" ./file.txt

  3. sed

    Find and replace 

              #Replace inline only first match per line.
          sed -i "s/search\(.*\)regex/replacement \1 text/" ./file.txt
          #Replace all matches. Not inline
          sed "s/search\(.*\)regex/replacement \1 text/g" ./file.txt

  4. kill

    kill a process by pid 

              #Get all processes | grab only targetProcesses | only pid column
          pid=$(ps -A | grep targetProcessName | awk '{print $1}')
          kill -9 $pid

  5. tr

    replace characters 

              #Replace tab with a space
          tr "\t" " "

  6. curl

    Pull some crap from the internet! 

              brushListUrl='http://alexgorbatchev.com/SyntaxHighlighter/manual/brushes/'
          brushRootUrl='https://agorbatchev.typepad.com/pub/sh/3_0_83/scripts/'

          brushes=$(curl "$brushListUrl" 2>/dev/null |
                    grep -oP "shBrush.*\.js")

          for brush in ${brushes[@]}
          do
            curl "$brushRootUrl$brush" > ./brushes/$brush
            sleep 1
          done
