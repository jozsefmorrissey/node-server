A collection token is used to verify a request has autorization to access passwords.

You can access your token with:
sudo confidentalInfo.sh value COLLECTION_IDENTIFIER token

For instance your token for the confidentalInfo collection is: {{token}}
You must pass this token to your application in order to access you passwords.

If you want some additional security you can set updateTokens=true, within a given collection and the application will update the token every time it is requested.


Note: Tokens are special in that you do not have to set them. If you request a token that does not exist. It will be generated.
